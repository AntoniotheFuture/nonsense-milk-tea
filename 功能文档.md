# 地狱级体验奶茶点单网页 - 功能文档

## 已完成功能

### 1. 登录/注册系统
- ✅ 支持20位以上复杂密码（大小写+数字+特殊字符）
- ✅ 用户信息存储在localStorage中
- ✅ 刷新页面会清除登录态（地狱级体验）
- ✅ 输错3次密码触发关不掉的错误弹窗

### 2. 首页 (Index)
- ✅ 每次进入强制观看10秒广告倒计时
- ✅ 底部导航栏（首页、点单、我的）
- ✅ 小概率随机错误弹窗（关不掉只能刷新）

### 3. 首页 (Home)
- ✅ 顶部用户头像和欢迎语
- ✅ 轮播广告（高度一致，但商品定制对话框有高度不一致轮播）
- ✅ 自取/外卖选择按钮
- ✅ 巨幅广告列表

### 4. 点单页面 (Order)
- ✅ 商品分类导航（奶茶、咖啡、果汁）
- ✅ 商品展示（经典奶茶、珍珠奶茶、拿铁咖啡、橙汁）
- ✅ 商品价格随机变动（每30秒70%概率）
- ✅ 点击商品打开定制对话框
- ✅ 购物车功能（底部按钮+抽屉式购物车）
- ✅ 购物车商品删除功能

### 5. 商品定制对话框
- ✅ 高度不一致的轮播图（导致页面跳动干扰操作）
- ✅ 甜度选择（不一致单位：Zero糖、25%糖、半糖、百分百糖）
- ✅ 数量选择
- ✅ 珍珠奶茶专属：可输入珍珠数量，生成对应数量的材质和颜色选择器（每颗都必选）
- ✅ 强制必填验证

### 6. 订单确认页面
- ✅ 显示商品清单和定制选项
- ✅ 根据自取/外卖显示不同信息
- ✅ 总计金额计算
- ✅ 付款按钮
- ✅ 地狱级警告：系统不会记录订单

### 7. 付款页面
- ✅ 付款成功动画
- ✅ 订单信息显示
- ✅ 查看订单详情按钮
- ✅ 返回首页按钮
- ✅ 地狱级警告：系统不会记录订单

### 8. 订单详情页面
- ✅ 自取：显示排队号、预计等待时间、当前服务号
- ✅ 外卖：显示配送状态、预计送达时间、骑手信息
- ✅ 商品清单和总计
- ✅ 地狱级警告：离开页面无法再次查看订单状态

### 9. 地狱级体验特色功能
- ✅ 复杂密码要求（20+位，包含各种字符）
- ✅ 刷新清除登录态
- ✅ 随机错误弹窗（关不掉）
- ✅ 商品价格随机变动
- ✅ 高度不一致轮播图（页面跳动）
- ✅ 不一致的甜度单位
- ✅ 每颗珍珠都要单独定制（强制必填）
- ✅ 系统不保存订单（需要用户截图）
- ✅ 离开订单详情页无法返回

## 技术栈
- Vue 3 + Composition API
- Vite 构建工具
- Element Plus UI 组件库
- Less CSS 预处理器
- 本地存储 (localStorage)

## 项目结构
```
src/
├── assets/           # 静态资源
├── components/       # 组件
│   ├── Ad.vue        # 广告组件
│   ├── NavButton.vue # 导航按钮
│   └── ProductCustomDialog.vue # 商品定制对话框
├── store/            # 状态管理
│   └── state.js      # 全局状态
├── views/            # 页面组件
│   ├── App.vue       # 根组件
│   ├── Index.vue     # 主页面框架
│   ├── Home.vue      # 首页
│   ├── Login.vue     # 登录/注册
│   ├── Order.vue     # 点单页面
│   ├── OrderConfirm.vue # 订单确认
│   ├── Payment.vue   # 付款页面
│   └── OrderDetails.vue # 订单详情
├── main.js           # 入口文件
└── style.css         # 全局样式
```

## 部署说明
- 使用 `npm run build` 打包
- 打包产物可直接托管在 GitHub Pages
- 包体积已优化，使用 Element Plus 按需引入
